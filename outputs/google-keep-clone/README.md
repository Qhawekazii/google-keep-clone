# Keep Clone

A Google Keep-inspired notes app built with plain HTML, CSS, and JavaScript.

## Features

- Keep-style top bar with logo, search, refresh, view toggle, and account button.
- Dark Google Keep-style layout.
- Light and dark mode toggle with saved preference.
- Functional menu button that toggles between icon-only and expanded sidebar modes.
- Left navigation for Notes, Reminders, Edit labels, Archive, and Bin.
- Compact "Take a note..." composer that expands while writing.
- Create notes with a title, body, and color.
- Display notes in Pinned and Others groups with a masonry-style card layout or a list-style view.
- Archive, unarchive, trash, restore, and permanently delete notes.
- Search notes by title or body.
- View note details in a modal.
- Tooltips on icon actions.
- Saves notes in the browser with `localStorage`.

Open `index.html` in a web browser. No build step or server is required.
